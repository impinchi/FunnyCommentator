What are Clusters?

Clusters are logical groups of ARK servers that typically:

  • Share the same network/host
  • Allow character transfers between servers
  • Have similar configurations
  • Are managed together
Host IP Guidelines:
  • 127.0.0.1 - Local machine
  • 192.168.x.x - Local network
  • 10.x.x.x - Private network
  • Public IP - Internet accessible
Tip: You can assign servers to clusters after creating them.
